How to Create an App Password for Outlook
Important: You need an App Password because regular passwords don't work with IMAP when 2FA is enabled.
Step-by-Step Instructions:
  1. Go to Microsoft Account Security:
    Visit: https://account.microsoft.com/security
  2. Sign in with your Outlook/UFL email and password
  3. Find "More security options" under the Security section
  4. Click "App passwords" in the left sidebar
  5. Click "Create a new app password"
  6. Give it a name like "EmailHelper Backup"
  7. Copy the 16-character password that Microsoft generates
    It will look like: abcd-efgh-ijkl-mnop
  8. Use this app password in the password field above (not your regular password)
Troubleshooting:
  • App passwords are single-use - create a new one if it doesn't work
  • Make sure you're using the generated password, not your regular one
  • If you don't see "App passwords", your account might not have 2FA enabled
Do I Have 2FA Enabled?
Quick Test:
  1. Visit: https://account.microsoft.com/security
  2. Sign in with your Outlook/UFL email
  3. If it asks for a verification code → You have 2FA → Need App Password
  4. If it lets you sign in directly → No 2FA → Use regular password
✅ No 2FA: Use your regular Outlook password in EmailHelper
⚠️ Has 2FA: Must create App Password (see instructions above)

Most university and corporate Outlook accounts have 2FA enabled for security.
